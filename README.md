## COVID-19 Data Analysis (BigQuery)

### Overview
This project analyses global COVID-19 data using **Google BigQuery**, focusing on population-level trends and vaccination progress.

### Dataset Information
**Tools:** Google BigQuery
**Data source:** Our World in Data (COVID-19)
**Coverage period:** 1 January 2020 - 23 November 2025
**Update status:** Latest available data at the time of analysis

### What this project covers
- Data exploration
- Vaccination trend analysis
- Aggregate Functions
- Window Functions
- Common Table Expressions (CTEs)
- SQL Joins
- Creating Views
- Data Type Conversion

### How to run
1. Upload datasets to BigQuery
2. Run SQL files in `/sql` folder in order
